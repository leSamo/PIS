Postup instalace

Závislosti

  • Node verzia 14
Spustenie servera + základné seedovanie: v priečinku BE/rest-api pomocou príkazu rm -rf target/ && nvm liberty:dev
Sputenie klienta: v priečinku FE pomocou príkazu npm install; npm run start
Sputenie cypress testov: v priečinku FE pomocou príkazu npm install; npm run test (testy vyžadujú nezmenenú pôvodnú datábazu)

Inicializácia databázy prebieha automaticky pri spustení Liberty servera.

Po spustení je možné okrem aplikácie pozorovať aj jednotlivé endpointy pomocu Swagger openAPI rozhrania na adrese localhost:9080/openapi/ui/
